## 📌 Избранные проекты

### 🚀 MP Neuro Bot — AI-ассистент для селлеров маркетплейсов
Telegram-бот для автоматизации создания визуального контента для маркетплейсов.  
Позволяет генерировать инфографику, нейрофотографии товара, варианты обложек для CTR-тестов, работать с тарифами, токенами и оплатами.

**Что реализовано:**
- генерация ТЗ и слайдов для инфографики
- генерация фото товара в разных ракурсах
- A/B варианты обложек для CTR
- тарифы и внутренняя валюта
- интеграция с YooKassa
- Redis-кэширование и FSM
- PostgreSQL + SQLAlchemy
- асинхронная обработка задач и защита от rate-limit

**Стек:** `Python` `Aiogram` `PostgreSQL` `Redis` `SQLAlchemy` `aiohttp` `OpenRouter` `YooKassa`

🔗 **Repository:** [mp-neuro-bot](https://github.com/DmitriyKholin/mp-neuro-bot)

---

### 🎨 Designer Booking Bot — CRM в Telegram для записи клиентов
Telegram-бот для автоматизации записи клиентов на услуги дизайна.  
Помогает принимать заявки, показывать свободные даты, собирать ТЗ, получать файлы от клиента и управлять расписанием.

**Что реализовано:**
- выбор услуги
- интерактивный календарь записи
- сбор ТЗ
- загрузка фото, видео и документов
- уведомления администратору
- подтверждение и отклонение заявок
- хранение данных в PostgreSQL
- автоматические напоминания через APScheduler

**Стек:** `Python` `Aiogram` `PostgreSQL` `SQLAlchemy` `APScheduler`

🔗 **Repository:** [designer-booking-bot](https://github.com/DmitriyKholin/designer-booking-bot)
